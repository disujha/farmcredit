# Demo Script - FarmCredit Connect

**Time Limit: 3 Minutes**

## Scene 1: The Problem (0:00 - 0:30)
*   **Narrator**: "Farmers in remote areas struggle to access credit because agents can't work without internet. Data is lost, and processing takes weeks."
*   **Visual**: Show the "OFFLINE" badge in the Client App.

## Scene 2: Offline Onboarding (0:30 - 1:00)
*   **Action**: 
    1. Open Client App.
    2. Click **+ New Farmer**.
    3. Enter details: "Ramesh Kumar", "Village: Rampur", "Land: 2.5 Acres".
    4. Click **Save**.
    5. Show "Pending Sync" indicator next to the name.
*   **Narrator**: "With FarmCredit Connect, agents can onboard farmers and capture data completely offline. It's stored locally and securely."

## Scene 3: Loan Request & Sync (1:00 - 1:45)
*   **Action**:
    1. Click **+ New Loan**.
    2. Select "Ramesh Kumar".
    3. Product: "Crop Loan", Amount: "50000".
    4. Click **Submit**.
    5. **TOGGLE NETWORK ON** (or simulate "Online" event).
    6. Watch the "Pending" badges turn into "Synced" (Green).
*   **Narrator**: "As soon as connectivity is restored, our Sync Manager orchestrates the data upload to the central server, resolving any conflicts automatically."
